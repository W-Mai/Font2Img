# Font2Img

A Freetype Playground.

# How to build

```shell
meson compile -C build
```

# Features

- [ ] Font loading and rendering.
    - [x] Load bitmap.
    - [ ] Load outline.
    - [ ] Load SVG
- [x] Font rendering in the console.
